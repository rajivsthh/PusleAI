# PulseAI — Brief

PulseAI is an AI-powered "News-to-Shorts" content engine that transforms complex global technology updates into short, engaging educational videos optimized for TikTok, Instagram Reels, and YouTube Shorts. It automates discovery, verification, script distillation, and multimedia generation to produce ready-to-post short-form videos within minutes.

Core workflow:
- Intelligence Agent: scans trusted sources (e.g., NASA, OpenAI Research, major tech publications) for high-impact developments.
- Verification Agent: fact-checks and validates content using Retrieval-Augmented Generation (RAG) to reduce hallucinations.
- Generation: distills verified findings into concise scripts, then produces synchronized visuals, subtitles, and natural-sounding voiceovers.

Target problems PulseAI addresses:
1. The Information Gap — making advanced technology news accessible and understandable.
2. The Production Gap — lowering cost and complexity of short-form video production.
3. The Engagement Gap — converting social media browsing into meaningful learning experiences.

Audience: students, self-learners, aspiring entrepreneurs, and early-career innovators in Nepal who need fast, digestible access to frontier technology trends.

Principles: responsible AI practices, real-time automation, and localized digital storytelling to democratize tech literacy and foster a technologically fluent generation.
